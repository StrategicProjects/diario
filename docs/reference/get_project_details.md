# Get details of a specific project

This function retrieves details of a specific project by its ID from the
Diario API.

## Usage

``` r
get_project_details(project_id)
```

## Arguments

- project_id:

  A valid non-empty string with the project ID.

## Value

A list containing the project details.

## Examples

``` r
if (FALSE) { # \dontrun{
project <- get_project_details("12345")
} # }
```
